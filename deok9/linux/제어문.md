# 제어문

## for
- 파일의 전체 라인을 한줄씩 출력

	-
	  ```
	  
	  line=10
	  for ((i=0 i<=line; i++)); do
	  sed -n ${i}p {test.txt}
	  done
	  ```

		- test.txt 파일의  0부터 10줄을 출력

## while
- 반복문을 통한 모니터링

	-
	  ```
	  while true; do df -h; sleep 3; done
	  ```

		- df -h 명령어를 3초에 한번씩 실행한다
